RemarkableSync

Backup your reMarkable tablet, convert notebooks to PDF, and transcribe handwriting to Markdown with AI — over USB or Wi-Fi.

View on GitHub Download

Features

🔌 USB & Wi-Fi Sync

Connect via cable or wirelessly over your local network. The config wizard can enable Wi-Fi SSH for you.

⚡ Incremental Backup

Only downloads files that have changed — tracked by size, modification time, and MD5 hash.

📄 PDF Conversion

Full support for v5 and v6 .rm formats with template backgrounds. Folder hierarchy preserved.

🤖 AI Handwriting Recognition

Send page images to GitHub Copilot or Claude to transcribe handwriting into clean text.

📝 Markdown Export

Each notebook becomes a .md file with YAML frontmatter and embedded page images.

👀 Watch Mode

Automatic periodic sync with a system-tray icon and optional run-at-startup.

How It Works

.rm files → SVG → PDF → PNG → AI OCR → Markdown

Your handwritten notes go from the tablet to searchable, AI-readable Markdown files — fully automated.
